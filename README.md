# WildFireVision 🛰️🔥
### Early Detection System via Satellite and Drone Imagery Fusion

## 📌 Project Overview
Wildfire outbreaks are increasing in frequency and destructive power. Detecting them at the earliest possible stage is critical to preventing large-scale ecological and economic damage. Traditional monitoring often suffers from latency or fails to cover remote, difficult-to-reach areas.

**WildFireVision** aims to build a robust monitoring framework that combines:
* **Smoke Detection:** Visual cues from optical sensors.
* **Thermal Hotspot Analysis:** Infrared signatures for high-confidence alerts.
* **Continuous Monitoring:** Temporal analysis to track fire evolution.
* **Multi-Source Integration:** Merging satellite data with localized drone feeds to reach blind spots.

## 🎯 The Mission
Our goal is to improve the **speed** and **accuracy** of wildfire alerts. By complementing global satellite data with agile drone deployment, we aim to provide authorities and communities with the necessary lead time to respond before fires spread.
